4  Working with Text

Working with text data in MS SQL Server involves various techniques and functions that allow you to manipulate, analyze, and clean textual information.

SUBSTRING:

Start & Length arguments

The SUBSTRING returns a specified portion of a string, starting at a specified position and with a specified length.

SQL
SELECT address, SUBSTRING(address, 5, LENGTH(address)) AS sub_string
FROM tableName
1
Get a sub of the address string starting from the 5 position to the last position.
address sub_string
Bwari, Abuja Abuja
Ikeja, Lagos Lagos
Note

  • The start parameter in SUBSTRING is 1-based, which means the first character is at position 1.

  • If the length is omitted in SUBSTRING it returns all characters from the start position to the end of the string.

  • If the length is greater than the remaining string length, the function returns the remaining characters.
