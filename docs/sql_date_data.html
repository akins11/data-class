5  Working with Date

5.1 Getting the current date and time

Current date

SQL
SELECT CURRENT_DATE AS current_date;
1
Returns the current date and time.
current_date
2024-09-05

Current Time

SQL
SELECT CURRENT_TIME AS current_time;
1
Returns the current date and time.
current_date
11:28:02.918290+01:00

Get Timestamp:

SQL
SELECT CURRENT_TIMESTAMP AS current_date_time;
1
Returns the current date and time (Timestamp).
high_precision_datetime
2024-09-04 11:29:05.220186

MAKE_DATE:

Creates a date from separate year, month, and day values:

SQL
SELECT 
  2023 AS year,
  12 AS month,
  31 AS day,
  MAKE_DATE(2023, 12, 31) AS constructed_date;
  

SELECT 
  2023 AS year,
  12 AS month,
  31 AS day,
  '2023-12-31'::DATE AS constructed_date;
year month day constructed_date
2023 12 31 2023-12-31




5.2 Extracting date and time parts

You can extract specific parts of a date or time using the following functions:

EXTRACT:

SQL
SELECT 
    CURRENT_TIMESTAMPAS current_date_time,
    EXTRACT(YEAR FROM CURRENT_TIMESTAMP) AS current_year,
    EXTRACT(QUARTER FROM order_timestamp) AS order_quarter,
    EXTRACT(MONTH FROM CURRENT_TIMESTAMP) AS current_month,
    EXTRACT(DAY FROM CURRENT_TIMESTAMP) AS current_day;
1
Extracts the year from the date.
2
Extracts the quarter from the date.
3
Extracts the month from the date.
4
Extracts the day of the month from the date.
current_date_time current_year current_quarter current_month current_day
2024-09-04 11:36:47.237779+01 2024 3 9 4

Others:

SQL
SELECT 
    TO_CHAR(CURRENT_TIMESTAMP, 'FMMonth') AS full_month_name,
    TO_CHAR(CURRENT_TIMESTAMP, 'Mon') AS abbrevaited_month_name,
    TO_CHAR(CURRENT_TIMESTAMP, 'FMDay') AS order_weekday_name;
full_month_name abbrevaited_month_name order_weekday_name
September Sep Wednesday




5.3 Date and time calculations

ADD:

Add a specified number of units (e.g., day, month, year) to a date.

SQL
SELECT 
  NOW() AS current_date_time,
  NOW() + INTERVAL '10 days' AS date_plus_ten_days,
  NOW() + INTERVAL '2 months' AS date_plus_two_months,
  NOW() + INTERVAL '1 year' AS date_plus_one_year;
current_date_time date_plus_seven_days date_plus_two_months date_plus_one_year
2024-09-05 14:25:04.569941 2024-09-12 14:25:04.569941 2024-11-05 14:25:04.569941 2025-09-05 14:25:04.569941

DATEDIFF:

Calculates the difference between two dates in a specified unit.

SQL
SELECT 
    '2023-12-31'::date AS date, 
    ('2023-12-31'::date - '2023-01-01'::date) AS days_difference;
date days_difference
2023-12-31 364




5.4 Date and time comparison

SQL
SELECT 
  CASE
    WHEN '2024-08-26'::DATE > '2024-08-01'::DATE THEN 'Later'
    ELSE 'Earlier or Same'
  END AS comparison_result;
comparison_result
Later
