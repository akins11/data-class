4  Working with Date

4.1 Getting the current date and time

SQL
SELECT GETDATE() AS current_date_time;
1
Returns the current date and time.
current_date_time
2024-08-31 02:45:08.504956

With higher precision:

SQL
SELECT SYSDATETIME() AS high_precision_datetime;
1
Returns the current date and time with higher precision.
high_precision_datetime
2024-08-27 02:15:08.8379183

ANSI SQL equivalent: to GETDATE()

SQL
SELECT CURRENT_TIMESTAMP AS CurrentTimestamp;
1
Returns the current date and time
high_precision_datetime
2024-08-27 02:20:53.373

DATEFROMPARTS:

Creates a date from separate year, month, and day values:

SQL
SELECT 
  2023 AS year,
  12 AS month,
  31 AS day,
  DATEFROMPARTS(2023, 12, 31) AS constructed_date;
year month day constructed_date
2023 12 31 2023-12-31

TRY_CONVERT:

This function converts a string to a date while handling conversion errors gracefully (returns NULL if conversion fails).

SQL
SELECT 
    TRY_CONVERT(DATE, '2024-08-26') AS try_converted_date_one,
    TRY_CONVERT(DATE, '2024-02-31') AS try_converted_date_two;
1
Convert the 26th of August to a DATE
2
Convert the 31st of February to a DATE
try_converted_date_one try_converted_date_two
2024-08-26 NULL

4.3 Extracting date and time parts

You can extract specific parts of a date or time using the following functions:

Date Parts:

SQL
SELECT 
    GETDATE() AS current_date_time,
    DATEPART(YEAR, GETDATE()) AS current_year,
    DATEPART(QUARTER, GETDATE()) AS current_quarter,
    DATEPART(MONTH, GETDATE()) AS current_month,
    DATEPART(DAY, GETDATE()) AS current_day;
1
Extracts the year from the date.
2
Extracts the quarter from the date.
3
Extracts the month from the date.
4
Extracts the day of the month from the date.
current_date_time current_year current_quarter current_month current_day
2024-08-27 02:41:44.380 2024 3 8 31
Other ways 
SQL
SELECT 
    GETDATE() AS current_date_time,
    YEAR(GETDATE()) AS current_year,
    MONTH(GETDATE()) AS current_month,
    DAY(GETDATE()) AS current_day,
current_date_time current_year current_month current_day
2024-08-27 02:41:44.380 2024 8 31

Weekday name:

SQL
SELECT DATENAME(weekday, GETDATE()) AS weekday_name;
weekday_name
Saturday




4.4 Date and time calculations

ADD:

Add a specified number of units (e.g., day, month, year) to a date.

SQL
SELECT 
  GETDATE() AS current_date_time,
  DATEADD(DAY, 10, GETDATE()) AS date_plus_seven_days,
  DATEADD(MONTH, 2, GETDATE()) AS date_plus_two_months,
  DATEADD(YEAR, 1, GETDATE()) AS date_plus_one_year;
current_date_time date_plus_seven_days date_plus_two_months date_plus_one_year
2024-08-31 02:45:11.853102 2024-09-07 02:45:11.853102 2024-10-31 02:45:11.853102 2025-08-31 02:45:11.853102

DATEDIFF:

Calculates the difference between two dates in a specified unit.

SQL
SELECT
    GETDATE() AS current_date_time,
    DATEADD(DAY, 7, GETDATE()) AS plus_seven_days,
    DATEDIFF(DAY, GETDATE(), DATEADD(DAY, 7, GETDATE())) AS days_difference;
current_date_time plus_seven_days days_difference
2024-08-31 02:45:12.259725 2024-09-07 02:45:12.259896 7
