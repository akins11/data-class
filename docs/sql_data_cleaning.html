6  Data Cleaning

6.1 Using ALTER AND UPDATE

Cleaning a table in SQL often involves updating data to correct errors, standardize formats, or remove unwanted information. In MS SQL Server, you can use the ALTER and UPDATE statements to clean and manage data effectively.

  • ALTER changes the structure of the table, which can help clean data indirectly by ensuring the columns are correctly defined.

  • UPDATE directly modifies the data, making it essential for correcting and standardizing the actual values stored in the table.

6.5 Splitting a combined column

Splitting a Combined Column into Multiple Columns.

SQL
-- Create and add the first_name & last_name columns to the table.
ALTER TABLE tableName
ADD first_name VARCHAR(100),
    last_name VARCHAR(100);

-- Splitting a full name into first and last names
UPDATE tableName
SET first_name = split_part(full_name, ' ', 1),
    last_name = split_part(full_name, ' ', 2);
  
SELECT full_name, first_name, last_name
FROM tableName;
1
Add a new column called first_name and first_name to the table.
2
Extract the first and last name from the full_name column.
3
Retrieve the full_name, and the extracted first and last names.

Removing duplicates records

Identify Duplicates

SQL
SELECT column1, column2, COUNT(*)
FROM tableName
GROUP BY column1, column2
HAVING COUNT(*) > 1;

Remove Duplicates

SQL
WITH duplicates AS (
    SELECT 
        column_id,
        ROW_NUMBER() OVER (PARTITION BY column1, column2, ORDER BY column_id) AS row_num
    FROM 
        tableName
)
DELETE FROM tableName
WHERE column_id IN (
    SELECT column_id FROM duplicates WHERE row_num > 1
);
